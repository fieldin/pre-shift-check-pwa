You're Offline

The Pre-Shift Check app needs to be loaded once while online before it can work offline.

To enable offline mode:

  1. Connect to WiFi or mobile data
  2. Open this app and wait a few seconds
  3. Set your name in Settings
  4. Now you can use it offline!

If you've already set up the app online, try closing and reopening this page.
